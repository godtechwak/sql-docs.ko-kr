# <a name="localdb_error_unknown_language_id"></a>LOCALDB_ERROR_UNKNOWN_LANGUAGE_ID
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>세부 정보  
  
| attribute | 값 |
| --------- | ----- |
|제품 이름|SQL Server|  
|이벤트 ID|270|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|지역화된 오류 메시지를 가져오는 중 오류가 발생했습니다. 'Language ID' 매개 변수에 지정된 언어를 알 수 없습니다.|  
  
## <a name="explanation"></a>설명  
 로컬 데이터베이스 런타임 오류 메시지에 대해 요청한 언어가 알 수 없는 언어이거나 지원되지 않는 언어입니다.  
  
## <a name="user-action"></a>사용자 동작  
 로컬 데이터베이스 런타임 오류 메시지에 대해 지원되는 언어 중 하나 또는 기본 언어를 요청하십시오.
